# fan_controller
Sets a Relay module on/off depending on several DHT?? sensors.

Components:
1. Arduino Micro
2. 1 2-channel Relay Module
3. 3 DHT22 temperature-humidity sensor
4. 3 10k ohm resisters; 1 for each sensor
